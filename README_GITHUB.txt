SITO HUMANITARNO ZDRUZENIE LABUNISHTA ITALIA

Pacchetto pronto per GitHub Pages.

ISTRUZIONI:
1. Apri il repository GitHub.
2. Carica tutti i file e le cartelle contenuti in questo ZIP nella root del repository.
3. Assicurati che index.html sia nella root.
4. Vai in Settings > Pages e pubblica da branch main / root.

NOTE:
- La pagina Trasparenza mostra informazioni sintetiche, non pulsanti di download.
- I PDF ufficiali sono conservati nella cartella documenti-interni come archivio, ma non sono linkati nel sito.
- Footer presente solo nella Home.
- Versione IT/MK inclusa.
