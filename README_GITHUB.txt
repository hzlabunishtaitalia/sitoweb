ISTRUZIONI PER GITHUB PAGES

1. Apri il repository GitHub del sito.
2. Carica tutti i file presenti in questa cartella nella root del repository.
3. Il file principale deve restare chiamato index.html.
4. Verifica che logo.jpeg rimanga nella stessa cartella degli HTML.
5. Vai su Settings > Pages e seleziona il branch main / root.
6. Dopo il caricamento, il sito sarà visibile dall’indirizzo GitHub Pages.

NOTE
- Design mantenuto dalla versione approvata.
- Footer visibile solo nelle pagine Home: index.html e index-mk.html.
- Testi italiani e macedoni aggiornati in stile più professionale e naturale.
- Per modificare una pagina, apri il relativo file .html e sostituisci solo il testo.
