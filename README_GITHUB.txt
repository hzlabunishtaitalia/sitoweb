ISTRUZIONI:
1. Apri il repository GitHub.
2. Carica tutti i file e le cartelle contenuti in questo ZIP nella root del repository.
3. Assicurati che index.html sia nella root.
4. Vai in Settings > Pages e pubblica da branch main / root.

- Footer presente solo nella Home.
